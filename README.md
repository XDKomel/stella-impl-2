# What expressions are supported?

### Base
* const true and false
* const nat
* const unit
* var
* application
* abstraction
* if
* equal
* sequencing (**new**)
* panic (**new**)
* referencing (**new**)

### Nat
* iszero
* succ
* natrec
* addition, subtraction, multiplication, division
* succ

### Tuple
* dot tuple
* tuple

### Records
* dot record
* record
* subtyping (**new**)

### Patterns
* inl, inr
* match
* let

### Lists
* list
* is empty
* head
* tail
